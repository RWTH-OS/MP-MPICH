::COMPILING MPICH APPLICATIONS:

Here are the settings necessary to compile an mpi application written in C/C++ with gcc.

Create a new makefile make the following settings:

include path:

If you installed the full source tree (mpich.nt.1.2.0.src.zip) If you installed the SDK (mpich.nt.1.2.0.sdk.zip)
  • [MPICH Home]\include
  • [MPICH Home]\mpid\nt_tcp_shm
  • [MPICH Home]\mpid\ch2
  • [MPICH Home]\mpid\util
  • [MPICH Home]\include

libraries:

  • libmpich_tcp_shm.a
  • libpmpich_tcp_shm.a
  • libmpich_tcp_shmd.a
  • libpmpich_tcp_shmd.a
  • Note: the pmpich*.lib libraries are only necessary if you make PMPI_ * calls in your code.

lib path:

  • [MPICH Home]\lib

Add your source files to the makefile and then build.
