Installation:

::REQUIREMENTS FOR DEFAULT INSTALLATION:

  • Administrator privileges on all the machines.
  • Administrative shares on each node (c$, d$ etc). (Note: the default NT installation provides these shares.)
  • All the machines you wish to install on using the gui wizard must be in the same Domain or accessable from the default network server. If you don't meet these requirements then the setup program will fail. If setup fails or you don't accept the defaults you can install manually. Manual Installation.

::INSTALLATION

(If you are using Winzip then simply click Install to do this automatically)

  1. Unzip the archive to a temporary location.
  2. Run Setup.exe to install/uninstall mpich. 
  3. Delete the temporary files you extracted.

::UNINSTALL

Run setup the same way you did in the Installation section above and select uninstall instead of install. If you no longer have the zip file, you can run setup from the [MPICH home]\bin directory. If you run setup from the [MPICH home]\bin directory and select complete uninstall, one file will be left on your hard drive - the setup program itself. You will have to delete it and the associated directory tree yourself.
