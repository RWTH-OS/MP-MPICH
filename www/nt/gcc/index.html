MPICH.NT.1.2.0

This release supports tcp/ip and shared memory connections. Interprocess communication on a single machine is done through shared memory queues and communication between processes on separate machines is done through sockets. This is not considered an optimal implementation.  It is a correct implementation which passes all but one function in our test suite and IBM's test suite .

Installation

1) Install the mpich dlls.

  • The runtime dlls come in the MPICH Runtime Dlls package or the MPICH SDK package.

2) Install a launcher.

  • MPICH Launcher: This is the preferred launcher.  It requires administrator privileges to install.
  • MPICH Simple Launcher: This launcher is provided for users who wish to evaluate MPICH without requiring any privileges to install.  It is limited in its capabilities.

3) Compile an MPI application

  • Compiling an MPI application

4) Run an application.

  • Each launcher provides an application launcher called MPIRun.exe.  Instructions can be found on the web pages provided with the launcher.
